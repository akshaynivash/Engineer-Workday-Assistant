Alternative Part Finder Chatbot



This chatbot is designed to help users find alternative parts for electronic components, such as fuses, based on their specifications. It integrates Generative AI (Phi-1.5) for physics-based explanations, Blenderbot for conversational interactions, and Llama 2 for meal planning. The chatbot also includes a personal assistant for schedule planning and uses Chroma DB for managing data.

Features
Alternative Part Recommendations:

Find alternatives for electronic parts based on descriptive analysis logic (tiered relaxation approach).

Generate physics-based explanations for the selected part and its alternatives using the Phi-1.5 model.

Personal Assistant:

Manage your schedule using Chroma DB.

Plan meals using the Llama 2 model.

Conversational Chatbot:

Use Blenderbot for simple and engaging conversational interactions.

Dynamic Logic and AI-Generated Disclaimers:

Use AI to dynamically adjust selection logic based on user requirements.

Generate context-aware disclaimers for relaxing constraints.

Setup
Prerequisites
Python 3.8 or higher

Libraries: streamlit, pandas, transformers, torch, chromadb, ollama

Hugging Face models: Blenderbot and Phi-1.5

Llama 2 model (pulled using Ollama)

Chroma DB account (for vector database)

Installation
Clone the repository:




pip install -r requirements.txt



Download models:

Blenderbot and Phi-1.5 from Hugging Face:


python -m transformers-cli download facebook/blenderbot-3B
python -m transformers-cli download facebook/phi-1.5
Place the downloaded models in the model/ directory.

Pull Llama 2 using Ollama:

Install Ollama (if not already installed):


curl -fsSL https://ollama.ai/install.sh | sh
Pull the Llama 2 model:


ollama pull llama2

Start the Streamlit app:


streamlit run app.py

Open the app in your browser:

The app will be available at http://localhost:8501.

Interacting with the Chatbot
Find Alternatives for a Part:

Enter the product ID of the part you need alternatives for.

Example:


User: Find alternatives for part A45.
View Physics-Based Explanations:

The chatbot will display the selected part’s details and a physics-based explanation generated by the Phi-1.5 model.

Explore Alternatives:

The chatbot will list alternative parts with justifications for their suitability.

Personal Assistant Features:

Use the chatbot to manage your schedule and plan meals.

User: Find alternatives for part A45 for an automotive application.

⚠️ **Disclaimer:** When relaxing constraints, ensure that the alternative part meets the temperature and durability requirements for automotive applications. Avoid using parts with lower breaking capacity or voltage ratings in high-power circuits.
